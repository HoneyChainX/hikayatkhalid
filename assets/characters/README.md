# assets/characters — locked character reference art (v1)

Front-facing, neutral, plain-background character model sheets for the recurring cast,
in the series style bible (`pipeline/characters.json`). Built to feed **Adobe Character
Animator's Characterizer** (auto-rig) for the puppet backbone — see `docs/PRODUCTION_HYBRID.md`.

| File | Character | Model | Notes |
|---|---|---|---|
| `khalid_ref.png` | Khalid (7) | higgsfield soul_2 (1536×2048) | ornate gold floral+geometric embroidery, kufi cap |
| `noor_ref.png` | Noor (5) | higgsfield soul_2 | soft-blue headscarf + dress (hair can be tucked) |
| `teta_ref.png` | Teta (grandmother) | higgsfield soul_2 | round glasses, cream hijab, olive-and-maroon |
| `khalid_ref_alt_nanobanana.jpg` | Khalid (alt style) | nano_banana_pro | taller proportions, simpler thobe — alternative look |

These are **v1 drafts** — regenerate/refine freely (cheap on soul_2, ~0.12 cr each). For rigging,
each needs its art separated into layers (mouth visemes, eyes/blink, brows, body, arms) or run
through Characterizer for an auto-rig. No prophets are depicted here (recurring cast only).
