# Shariah Review Packet — ep02–ep10

*For the reviewing sheikh. Purpose: approve the `source_text` for each episode so
it can enter production. Per the project's hard rules, **no image/audio/script
generation happens before this approval**, and the AI **only** dramatizes from
the approved `source_text` — it never adds religion from its own "knowledge."*

> ⚠️ **AI did not write the scripture here.** Each "proposed story arc" below is a
> plain-language outline for you to **verify or correct**. The **verbatim Qur'an
> verses / hadith wording must be inserted by you from primary sources**
> (Mushaf; Ṣaḥīḥ al-Bukhārī / Muslim) — do not rely on the model for exact sacred
> text. For the 5 prophet episodes, **no prophet is depicted** (light / voice /
> off-camera only); the on-screen "heroes" are the non-prophet elements.

**Process:** you fill *"النص المعتمد (source_text)"* and tick the checklist →
the dev runs the SQL at the bottom to write it into `backlog.source_text` and set
`status='planned'→ready` → Workflow A picks it up behind the Telegram shariah
gate.

---

## القاعدة (تذكير)
1. **لا تجسيد لأيّ نبيّ** إطلاقاً (نور/صوت/خارج الكادر فقط).
2. المحتوى الديني يُؤخذ **حرفياً** من المصدر المعتمد، لا من «معرفة» النموذج.
3. ملخّص القصّة أدناه **مقترح للمراجعة** — يُعتمد اللفظ النهائي من المصدر الأصلي بخطّكم.
4. مناسب للأعمار ٤–١٠، بلغة عربية مبسّطة، دون ترويع.
